# 30-Day SEO Rollout Checklist

## Week 1 - Technical Baseline
- [ ] Verify `robots.txt` and `sitemap.xml` are live on production
- [ ] Submit sitemap in Google Search Console
- [ ] Ensure all key pages have unique title + meta description
- [ ] Set up GA4 conversion events (sign-up, upload, checkout)
- [ ] Add UTM naming convention for campaigns

## Week 2 - Landing Page Quality
- [ ] Improve copy depth on `/nav-hatarozat-ertelmezes` to 900+ words
- [ ] Improve copy depth on `/szamla-ocr` to 900+ words
- [ ] Improve copy depth on `/dokumentum-archivum` to 900+ words
- [ ] Expand `/arak` and `/pricing` with intent-aligned copy blocks
- [ ] Add customer proof/mini case snippets (if available)

## Week 3 - Blog Authority
- [ ] Publish 3rd and 4th articles in `/blog`
- [ ] Add internal links from each post to one product page
- [ ] Add at least one internal link from product pages back to blog
- [ ] Refresh FAQ with 2 new questions from support tickets

## Week 4 - Measurement and Iteration
- [ ] Review Search Console: impressions, CTR, average position by URL
- [ ] Rewrite titles/descriptions for low CTR pages
- [ ] Identify top 5 growing queries and create matching content
- [ ] Plan next 4 content pieces from query gaps
- [ ] Record baseline vs. end-of-month conversion metrics

## KPIs to Track
- Organic sessions (weekly)
- Non-brand impressions and clicks
- CTR by landing page
- Sign-up conversion rate from organic traffic
- Upload/activation rate from organic sign-ups
